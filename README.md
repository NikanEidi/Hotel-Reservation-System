# Grand Plaza Hotel Reservation System

A complete management system for the Grand Plaza Hotel built on JavaFX and MySQL database.

## System Features
*   **Fully Functional Kiosk Booking System**: Including Room Selection, Add-ons, Billing summary with PDF export, and database saving.
*   **Secure Administration**: Login via hashed credentials using JBcrypt.
*   **Admin Dashboard**: Real-time business metrics utilizing Hibernate ORM models tracking Check-Ins, Available Rooms, and total Revenue.
*   **Live Database**: Seamlessly integrated to an active MySQL database handling thousands of potential active bookings dynamically.

## Installation
1.  Run the Maven Wrapper package build: `./mvnw clean package`
2.  Start the Application: `./mvnw javafx:run`
3.  Login via the hidden secure login (Top Right). Admin Username `nikan`, Password `123456`.
